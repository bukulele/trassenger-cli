Trassenger 0.3.1 — Windows ARM64
=================================

INSTALLATION
------------
1. Extract all files from this archive to a folder of your choice,
   for example: C:\Users\You\Trassenger\

2. The folder must contain both files:
     trassenger-daemon.exe
     trassenger-tui.exe

3. Run trassenger-daemon.exe

   A Trassenger icon will appear in the system tray (bottom-right corner).
   Click it to open the messenger.

AUTOSTART
---------
Right-click the tray icon and enable "Start at Login" to have the daemon
start automatically when you log in.

UNINSTALL
---------
1. Right-click the tray icon → Quit
2. Delete the folder with the two .exe files
3. Optionally delete app data: %APPDATA%\trassenger\
